Condense

Condense is a Minecraft font that is narrower than Minecraft’s default font. When using this font, text will often take up less space, and you may find that you can read text more quickly.

Download

  • Latest version, v0.2.2 for Minecraft 1.17
  • v0.2.1 for Minecraft 1.16.2 to 1.16.5
  • v0.2.0 for Minecraft 1.15 to 1.16
  • v0.1.2 for Minecraft 1.15 to 1.16
  • v0.1.1 for Minecraft 1.15 to 1.16
  • v0.1.0 for Minecraft 1.13 to 1.14

Installation

On Minecraft 1.16.2 and higher, go to Options > Resource Packs, then drag the zip file into the Minecraft window.

For older versions, see See Tutorials/Loading a resource pack on the Minecraft Wiki.
